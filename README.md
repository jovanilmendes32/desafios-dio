# Desafio Criativo - Engenharia de Prompt

## Sobre

Este repositório contém minha solução para o Desafio Criativo da DIO sobre Engenharia de Prompt.

## Objetivo

Desenvolver um prompt estruturado para analisar feedbacks de clientes bancários utilizando técnicas de Engenharia de Prompt, incluindo:

- Instruções claras
- Contexto
- Few-shot Learning
- Guardrails
- Formato de saída
- Conteúdo de suporte
- Divisão da tarefa
